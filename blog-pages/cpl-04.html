CHARACTERISTICS OF C++


C++ is not a purely object oriented language, but a hybrid that contains the functionality of the C programming language. This means that you have all the features that are available in C.

What is OOPs(Object -Oriented Programming)



Object-oriented programming (OOP) is a programming language which is organized by objects rather than actions and data. An object-oriented programming characterized by data controlling access to code of OOPS.
