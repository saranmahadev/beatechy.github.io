C++ is a general-purpose programming language and widely used nowadays for competitive programming. It has an imperative, object-oreinted and generic programming features. C++ runs on lots of platform like windows, linux, unix, mac, etc.

what is C++?



C++ is a general purpose, case sensitive free from programming language that supports object oriented, procedural and generic programming.

Object-oriented programming(OOPs)



The four major pillars of object-oriented programming(OOPs) used in C++ are

1) Inheritance
2) Polymorphism
3) Encapsulation
4) Abstraction

C++ Standard Libraries



Standard C++ programming is divided into three important parts:

1. Core library (data types, variables and literals, etc.)
2. Standard library (set of function manipulation strings, files, etc.)
3. Standard Temple Library(STL)-[set of methods manipulating a data structure]
