STRUCTURE OF FUNCTION MAIN()



C++ is an object oriented programming language which accompanying member function and global function. Simply the function having its own particular task and it doesn’t belong under any single particular class.

You can create your own function by your knowledge, otherwise you can use the ready made function from standard libraries.
And it is necessary to write the global function “main()” it the has pays a special role do to the program. While starting a program we must use a symbol ‘#’ which indicates that the line of the program is indented for the preprocessor. The preprocessor is just one in the translation phase and no object code is there at the time.

#include< filename.h >



This allows the program to access all information from he header file. The header file isostream comprises of conventions for input stream and output streams. The word stream represents that the information will be treated as a flow of data.
Predefined names in C++ are to be found in the std (standard) namespace. The using directive allows direct access to the names of the std namespace.
Program execution begins with the first in the function main() and this is why the c++ program have function main(). The name cannot be changed, this function’s structure is not different from that of any other C++ function.

Let we take an example in the function main()
cout << "LEARN C++ PROGRAM FROM BE A TEACHY" << endl;

The word cout(console output) which is responsible for an object to give an output. The two less-than symbols, <<, indicate that characters are being “pushed” to the output stream. Finally endl (end of line) causes a line feed. The statement return 0;
Terminates the function main() and also the program, returning a value of 0 as an exit code to the calling program. It is standard practice to use the exit code 0 to indicate that a program has terminated correctly.
Note that statements are followed by a semicolon. By the way, the shortest statement comprises only a semicolon and does nothing.
