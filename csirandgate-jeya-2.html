Quantum Mecanics-WaveFunction



Let us learn to identify the admissible wavefunction(Ψ) to describe the wave nature of a particle or a system. The wavefunction should satisfy normalization condition which implies that Ψ becomes zero as x tends to infinity. It must be a single-valued function and the value should not approach infinity at any point. Also it should be continous. Let us make use of these informations to identify a qualified wavefunction.
